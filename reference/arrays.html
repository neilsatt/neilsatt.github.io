Array methods

Original Array var names = ['ted', 'lisa', 'veronica', 'enrique']

var nameLength = names.length - length gives the total number of elements in an array

names.push('alice') - Push adds another element to array. The return value is the new length of the array.

Length of original array: 4
Updated names array after names.push: ted,lisa,veronica,enrique,alice
names.push return value: 5

names.pop() - Pop removes the last element from the array. The return value is the element that was removed.

Updated names array after names.pop: ted,lisa,veronica,enrique
names.pop return value: alice

names.unshift('francis') - Unshift adds one or more elements to the beginning of the array. The return value is the new length of the array.

Updated names array after names.unshift: francis,ted,lisa,veronica,enrique
names.unshift return value: 5

names.shift() - Shift removes the first element of the array. The return value is the element that was removed.

Updated names array after names.unshift: francis,ted,lisa,veronica,enrique
names.unshift return value: 5

names.reverse() - Reverse reverses the order of elements.

Updated names array after names.reverse: enrique,veronica,lisa,ted

names.sort() - Sort sorts the elements into ascending/alphanumeric order. You can also pass a sortFunction that defines an alternative sort order.

Updated names array after names.sort: enrique,lisa,ted,veronica

names.splice(1, 2) - Splice removes elements. The start index is the 1st parameter and the number of elements to remove is the 2nd parameter.
The return value is the elements that were removed.

Array before splice: enrique,lisa,ted,veronica
Updated names array after names.splice: enrique,veronica
names.splice return value: lisa,ted

names.slice(2, 4) - Slice returns a new array. The start index is the 1st parameter and the 2nd parameter is the number where you stop removing.

Array before slice: ted,lisa,veronica,enrique
names.slice returns a new array
names.slice return value: veronica,enrique

var newArray = ['bob', 'roddy', 'billy', 'stacy'];

names.concat(newArray) - Concat returns a new array that consists of the original array plus any arrays in the parameter list.

Array before names.concat: ted,lisa,veronica,enrique
Array after names.concat: ted,lisa,veronica,enrique,bob,roddy,billy,stacy

names.join('*') - Join converts all of the array elements to strings. The parameter is for when you want something other than a comma to separate the elements.

Array before names.join: ted,lisa,veronica,enrique
Array after names.join: ted*lisa*veronica*enrique
