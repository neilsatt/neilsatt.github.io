Objects

Creating an Object
var bankStatement = {};

Creating a custom Object with a property and method
var bankStatement = {
  interest: 2.75,
  myBalance: function (total, interest) {
     return total * interest;
   }
};

Using dot notation to add a property and method
bankStatement.holidayBonusRate = 3.5;
bankStatement.calculateBonusRate = function (total, holidayBonusRate) {
     return total * holidayBonusRate;
   }
};

Constructor Functions - with properties, and parameters
var BankStatement = function(balance, taxRate) {
  this.balance: 200.00,
  this.interest: 2.75,
  this.taxRate = taxRate;
 }

Adding a method to the prototype
BankStatement.prototype.showTotal = function(balance, total) {
     return balance * total;
   }
};

Prototypes
Prototypes are objects that are cloned to create a new object. This is known as prototypal inheritance.

You should create methods on the object's prototype if you are going to create multiple instances of an object.
This saves memory since each object instance refers to the prototype method, instead of creating a duplicate method of its own.

Prototype Methods:
 toString - returns a string representation of the object.
 valueOf - returns a primitive value of the object.
 hasOwnProperty - returns a Boolean value that indicates whether a property is defined on the object or the object's prototype.

Additional Object methods, operators and terminology

The create method
The create method lets you specify which object to use as an object's prototype.

var originalPrototype = {};
var newPrototype = Object.create(originalPrototype); // inherits from the original prototype

Factory Function
Functions that create, initialize, and return objects.

Inheritance - lets you create a parent object with common methods that you can then extend with more specialized methods.
To inherit the methods of an object created by a constructor, set the prototype object of the child object to an instance of the parent object.

var ContructorOne = {};
var ConstructorTwo = {};
ConstructorTwo.prototype = new ConstructorOne();

Cascading/Chained method
A method that can be chained with other methods. A method must return the object represented by the this keyword if you want it to be chained with other methods.

Operators:
 in - returns true if an object has the named property.
 instanceof - returns true if an object is an instance of the specified type. Includes the object typeof the object and an inherited object type.
 typeof - lets you determine the type of a variable:
   - number, string, Boolean
   - object (objects and nulls)
   - function (functions and constructors)
   - and undefined
