Functions

Function expression - assigned to a variable and it is referred to by that variable name. Sometimes called an anonymous functions since they don't have names. Often used to handle events.

var myFunction = function() {
   // function statements here
};

Must be created before it can be called.

Function declaration - not stored in a variable, and they are declared after the function keyword.

function myFunction () {
   // function statements here
};

Function declarations are hoisted to the top of the context, which means they can be called before they are created in the code.

Scope

Scope - refers to where a variable or function is accesssible.

Local Scope - when variables are created inside a function. They can only be referred to by code within that function.

Global Scope - when variables are created outside of a function or without the var keyword. All code has access to these variable, so they generally aren't recommended.

Lexical Scope - this means you can determine the scope of a variable or function by looking at where it is in the code.

Terminology

First-class function - is a term that means that functions can be used like any other value in JS including, being able to be passed as an argument, returned from a function, or assigned to a variable.

Pass by value - is how arguments are passed to functions. In otherwords, they are passed by copy, which means the original value does not get changed.

this - refers to the object that the method is bound to.

Built in Function methods that manipulate this

call - executes the function with a particular this value and with specific parameters.
  1) The first parameter is the value to which this should be equal when the function is executed
  2) Any other parameters are the ones that should be passed into the function

apply - works the same as the call function, but it only accepts two paramenters:
  1) The value of this
  2) An array or array like object of parameters to pass to the function

bind -
  1) The this value for the new function
  2) All other arguments represent named parameters that should be permanently set in the new function

IIFE

Immediately Invoked Function Expression - When a function is defined and invoked at the same time by adding parentheses at the end. The entire function is also wrapped inside parentheses.

(function () {
   // function statements here
}) ();
