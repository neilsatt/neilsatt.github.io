Prototypes


A Prototype is an object used that is cloned to create a new object.

- Methods on a prototype object are available to every object that uses it

- Once you create an instance of an object, any property you add will be added directly to the object, not the object's prototype

Adding a method to an object using the prototype property
Car.prototype.drive = function(speed) {
   // drive statements here
};

Methods of a new object's prototype object
toString - Returns a string that represents the object

valueOf - Returns the primitive value of the object

hasOwnProperty - Returns a Boolean value that indicates if a specific property is defined on the object, rather than on the object's prototype.
