Closures

Closure - an inner function that has access to an outer function's variables. They are created when a reference is made to a
  1) Variable
  2) Object
  2) Function that is defined within another function

One common technique for closures is for an outer function to return the inner function.

Closures can be used to create private state.

Basic Closure Example
The farther inside you are in a function, the more variables you have access to
var g = globalVar = "Global Variable";
//can only access 'g' at this level

var outerFunction = function() {
  var o = "Outer Variable";
   //can access 'g' and 'o' at this level

   var innerFunction = function() {
      var i = "Inner Variable";
      //can access 'g' and 'o' and 'i' at this level
    };
   };

Counter Closure Example
One common closure example is when the outer function returns the inner function
var createClickCounter = function() {
   var count = 0;
   var clickCounter = function() {
      count++;
      console.log(this.id + 'count is" +count);
   };

   return clickCounter; // keeps the inner function's variables 'alive'
};

window.onload = function() {
  //each call to createClickCounter creates a closure
   $('btn1').onclick = createClickCounter();
   $('btn2').onclick = createClickCounter();
};

Each btn will keep track of its own unique counter
  // Sample output
   btn1 count is 4
   btn2 count is 3
