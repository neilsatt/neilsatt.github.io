Below is an example of absolute positioning. This time we still have 3 divs with the same width, but the middle one uses absolute positioning:

  1. The second paragraph has been set to position:absolute
  2. The paragraph has been moved 500 pixels to the right. Absolutely positioned elements are moved relative to the nearest positioned ancestor/parent
  3. Notice that the space it would have occupied without using absolute positioning, has now been closed up. This means the div has been taken out of the main flow.
