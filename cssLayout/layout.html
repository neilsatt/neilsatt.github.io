One of the most common ways to layout pages is using the float property. The float property is often used to position an image to the left or right side, but you can also use it to create columns.
Below is an example of a simple two column layout.

  1. For the left side, use float: left and set the width property to around 20%
  2. Set the main content to have the margin-left property set somewhere around 20%
  3. Set the padding to an amount you like
