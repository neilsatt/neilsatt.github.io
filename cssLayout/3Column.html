A basic three column layout example:

  1. Use float: left for the left side, and set its width to 20%
  2. Use float: right for the right side, and set its width to 20% also
  3. For the main content, set its width somewhere between 55-60%, depending on how much margin has already been set for the overall page
  4. Add padding to all three sections to refine the overall look
  5. User clear: both after the 3 sections if you decide create a footer
