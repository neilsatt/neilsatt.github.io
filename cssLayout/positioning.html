The four most common CSS position properties used are: static, relative, absolute, and fixed.

  • Static is the default
  • An element with a fixed position won't move when the browser is scrolled. Try scrolling and notice how the box below stays in the same place.
  • A relatively positioned element is positioned relative to its current position. If there is any surrounding content, it doesn't move, so a gap will exist.
  • An absolute positioned element gets completely removed from the flow. Any surrounding content will take up the space previously used by the element.
  • Both relative and absolute positioning use the top, botton, left, and right properties to move elements. Check out the links in the drop down menu for examples of each.
